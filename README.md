# Python Mathematical Programming Exercises

This repository contains solutions to several programming exercises implemented in Python.  
The problems mainly focus on mathematical computations and testing properties of integers using basic programming concepts such as loops, functions, and conditionals.

The purpose of this repository is to practice problem solving and understand how mathematical ideas can be implemented using Python.

## Contents

Problem 1 – Greatest Common Divisor (GCD)
This program calculates the Greatest Common Divisor (GCD) of two positive integers entered by the user.  
The program works by finding all the factors of each number, identifying the common factors, and then selecting the largest one as the GCD.

Basic error handling is also included to deal with invalid or negative inputs.

Problem 2 – Approximating Euler’s Number (e)
This problem explores different ways of approximating Euler’s number (e).

The following approaches are implemented:
- Using Euler’s limit formula
- Using the infinite sum expansion of e
- Computing approximation with a tolerance value (epsilon)
- Printing the error for both methods for different values of k

The program also compares the accuracy of the two approximation methods.

Problem 3 – Testing Integer Properties
This program checks certain properties of integers.

Functions included:
- `is_power_of_2(n)` – Determines whether a given number is a power of 2.
- `is_power(b, n)` – A generalized version that checks whether `n` is a power of another integer `b`.

These functions rely only on division and modulus operations.

 Technologies Used
- Python 3
- Basic Python libraries such as `math`

 Purpose of the Repository
This repository was created as part of coursework to practice Python programming and strengthen understanding of mathematical algorithms.
